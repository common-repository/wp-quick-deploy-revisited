== Description ==

WP Quick Deploy Revisited allows you to pick your favorite plugins and install them at once without hassle. 
WP Quick Deploy Revisited is one of the easiest ways to get the essential SEO plugins loaded after a fresh install of WordPress. 
There's no need to manually hunt for them again, you can simply install them in bulk.
It also allows you to activate/deactivate, delete and update your plugins. Everything in one plugin!

In the revisited version you can also manage your own set of plugins. Export it and import it in a new installation.

Created by Valentin Mayr, in large parts based on the work of Vladimir Prelovac.

== Installation ==

1. Upload the whole plugin folder to your /wp-content/plugins/ folder.
2. Go to the Plugins page and activate the plugin.
3. Go to WP Quick Deploy Revisited settings to start installing plugins

== Frequently Asked Questions ==

= I have problems installing new plugins =

WP Quick Deploy Revisited currently relies on file permissions set up correctly on your server, in order to write files to your plugin folder. Some servers require ftp login details to write the files and Plugin Central will support this option in the future.
